Использование: %s [ОПЦИИ]

Генерирует команды composer require из "composer outdated --direct".
Работает с любым PHP-проектом (Symfony, Laravel, Yii, CodeIgniter, CakePHP, Laminas, Slim, и т.д.)

ОПЦИИ:
    --run                    Выполняет предложенные команды автоматически
    --release-info           Показывает информацию о версии (краткое описание со ссылками)
    --release-detail         Показывает полный changelog для каждого пакета (подразумевает --release-info)
    --no-release-info        Пропускает раздел информации о версии (поведение по умолчанию)
    -v, --verbose            Показывает подробный вывод (файлы конфигурации, пакеты, и т.д.)
    --debug                  Показывает отладочную информацию (очень подробно, включает пути к файлам, парсинг, и т.д.)
    -h, --help               Показывает это сообщение справки

ПРИМЕРЫ:
    %s                                    # Показывает предложенные команды (без информации о версии по умолчанию)
    %s --run                              # Выполняет предложенные команды
    %s --release-info                     # Показывает краткое описание информации о версии
    %s --release-detail                   # Показывает полные changelog (подразумевает --release-info)
    %s --verbose                           # Показывает подробный вывод с деталями конфигурации
    %s --debug                             # Показывает подробную отладочную информацию
    %s --run --release-detail             # Выполняет и показывает полные changelog
    %s --verbose --release-info # Подробно с информацией о версии
    %s --debug                  # Режим отладки (очень подробно)

ПОДДЕРЖКА ФРЕЙМВОРКОВ:
    Скрипт автоматически учитывает ограничения версий фреймворка:
    - Symfony: учитывает ограничение "extra.symfony.require"
    - Laravel: учитывает версию major.minor laravel/framework
    - Yii: учитывает версию major.minor yiisoft/yii2
    - CakePHP: учитывает версию major.minor cakephp/cakephp
    - Laminas: учитывает версии major.minor laminas/*
    - CodeIgniter: учитывает версию major.minor codeigniter4/framework
    - Slim: учитывает версию major.minor slim/slim

ИГНОРИРУЕМЫЕ ПАКЕТЫ:
    Пакеты, перечисленные в generate-composer-require.yaml или .yml, будут пропущены.
    Скрипт ищет файлы конфигурации в текущей директории (где находится composer.json).
    Старый формат (generate-composer-require.ignore.txt) по-прежнему поддерживается для обратной совместимости.
    Комментарии, начинающиеся с #, игнорируются.

ИНФОРМАЦИЯ О ВЕРСИЯХ:
    По умолчанию информация о версиях НЕ показывается (API-вызовы не выполняются).
    Используйте --release-info, чтобы показать краткое описание с:
    - Именем пакета
    - URL версии
    - URL changelog

    Используйте --release-detail, чтобы увидеть полный changelog версии.
    Используйте --no-release-info, чтобы явно пропустить информацию о версиях (поведение по умолчанию).
